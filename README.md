# Ma3an

# Project Overview
Ma3an is a web-based travel management platform designed to simplify the organization and operation of guided tours.
The system connects travel agencies, tour guides, and travelers in one centralized platform, ensuring smooth coordination and clear role separation.

# Problem Statement
Travel agencies often face challenges in managing tours, assigning available tour guides, and keeping travelers informed. Also communication with travelers is often fragmented or manual. Additionally, tour guides cannot effectively track travelersâ€™ locations, which may lead to travelers getting lost or losing their way during tours.

# Our Solution
Ma3an provides dashboards for agencies, guides, and travelers, simplifies tour management, enables communication with travelers, and includes real-time location tracking with geofencing.

# Key Features:
- An easy-to-use dashboard for travelers, tour guides and agencies
- Tour creation, assignment, and tracking
- Tour schedule creation, assignment, and tracking
- Both traveler and tour guide can view tour details through dashboard
- Communication tools for tour guide with travelers
- Real-time location tracking
- Geofencing: alerts when travelers leave designated areas
- Traveler can leave a review after the tour ends
